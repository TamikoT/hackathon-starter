Muviato-API

Available Endpoints

Users

Acceptable Params:

  • username: string
  • isHost: boolean

GET /users returns all users

POST /users - creates a new user

GET /users/:id finds an user by id

PUT /users/:id updates existing user

DELETE /users/:id to delete a room by id

Rooms

Acceptable Params:

  • code: string
  • _hostID: string

GET /rooms returns all rooms

POST /rooms - creates a new room

GET /rooms/:id finds a room by id

PUT /rooms/:id updates existing room

DELETE /pets/1 to delete a room by id
