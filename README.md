### *Setup*
1. Run  `git clone https://github.com/TamikoT/movie-chat-app`
1. Run `npm install`
1. Run `npm start`

### *Tech Used*

| NAME	       | DESCRIPTION |
|--------------|-------------|
| ReactJS      | views |
| React Router | universal routing |
| Redux	       | state management |
| MongoDB	     | NoSQL database |
| Express	     | web server framework |

| Webpack	     | bundle + transpile |
